# **App Name**: ExamBridge

## Core Features:

- URL Analysis: Analyze the syllabus and module structure of a given paid learning resource URL.
- Exam Category Selection: Allow users to select an exam category to refine resource matching.
- Resource Matching: Find free alternative resources that match the topics in the paid resource using the Gemini API as a tool.
- Study Path Generation: Generate a personalized 'Study Path' dashboard for the user.
- Module Display: Display each module with its topic, free resource link, and a rationale for the match.
- User Authentication: Enable user sign-up/login using Firebase Authentication.
- Dashboard Saving: Save generated dashboards to the user's profile using Cloud Firestore.

## Style Guidelines:

- Primary color: Electric Blue (#3B82F6) for a modern, tech-focused feel.
- Background color: Deep Navy (#0F172A) for a premium SaaS dark mode aesthetic.
- Accent color: Teal (#43c6ac) to complement the electric blue
- Body and headline font: 'Inter' (sans-serif) for a clean and professional look. Note: currently only Google Fonts are supported.
- Use modern and minimalist icons to represent different exam categories and study modules.
- Implement a card-based layout with rounded corners and subtle shadows for a premium SaaS feel.
- Use subtle animations for loading states and transitions to enhance user experience.
